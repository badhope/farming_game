## 🌾 项目唠唠嗑
这玩意儿本质上是个「新手村级」农场模拟游戏骨架，主打一个**练手+瞎折腾**！目前代码就像刚撒完种的菜地，就冒了几根芽——不是不想肝，实在是打工人的时间被老板/生活按在地上摩擦😮‍💨。

核心就扒拉了点基础模块：
- 时间系统（种地哪能没白天黑夜！）
- 作物/地块模型（总不能空气种地吧）
- 简易经济系统（卖菜换钱才是种地的终极浪漫）
- 存档/显示啥的边角料

## 🛠️ 目前啥状态？
主打的就是一个「有骨架没肉」：
- 代码能跑，但功能约等于「只能在地里刨个坑」
- 文档稀碎，全靠注释续命
- 玩法？不存在的，现阶段就够自己练手用

## 📁 项目结构（目前就这点家当）
```
farming_game/
├── LICENSE          # 许可证（反正就是随便造）
├── README.md        # 你现在看的这个玩意儿
├── main.py          # 入口文件（能跑就行版）
├── requirements.txt # 依赖（暂时就空的，主打一个轻量）
├── config/         # 配置文件（瞎写的）
├── systems/         # 核心系统（时间/存档/显示）
├── core/            # 核心逻辑（经济/游戏管理）
└── models/          # 数据模型（作物/玩家/地块/成就）
```

## 🚀 怎么跑起来？
1. 克隆仓库：`git clone https://github.com/你的账号/farming_game.git`
2. 进目录：`cd farming_game`
3. 装依赖（如果有的话）：`pip install -r requirements.txt`
4. 启动游戏：`python main.py`
