## 游戏简介

农场大亨是一款轻松有趣的农场经营模拟游戏，玩家可以在游戏中体验种植作物、养殖动物、经营农场的乐趣。游戏支持多平台游玩，你可以随时随地在网页端或桌面端继续你的农场之旅。

### 游戏特色
- 🌾 **丰富作物**：45+种作物可供种植
- 🏆 **成就系统**：40+个成就等你解锁
- 🛒 **商店系统**：80+种商品供你选购
- ☁️ **云端同步**：游戏进度自动保存到云端
- 📱 **跨平台**：支持网页端和桌面端

## 开始游戏

### 注册账号
1. 打开游戏网页
2. 点击「注册」按钮
3. 填写用户名、邮箱和密码
4. 点击「注册」完成注册

### 登录账号
1. 输入用户名和密码
2. 点击「登录」
3. 开始你的农场之旅

### 游客模式
如果你不想注册账号，也可以点击「游客模式」直接开始游戏。游客模式下的数据存储在本地，重装浏览器会导致数据丢失，建议注册账号以保存进度。

## 核心玩法

### 1. 种植作物

#### 播种
1. 点击空置的土地
2. 选择要种植的作物种子
3. 确认种植

#### 浇水
- 点击已种植的土地进行浇水
- 浇水后作物会加速生长
- 每天需要重新浇水

#### 收获
- 作物成熟后会显示成熟标志
- 点击成熟作物进行收获
- 收获获得金币和经验值

### 2. 季节系统

游戏中有四个季节，每个季节适合种植不同的作物：

| 季节 | 适合作物 |
|------|---------|
| 🌸 春季 | 草莓、樱桃、生菜、豌豆 |
| ☀️ 夏季 | 西瓜、番茄、玉米、葡萄 |
| 🍂 秋季 | 南瓜、茄子、苹果、菊花 |
| ❄️ 冬季 | 大白菜、胡萝卜（温室） |

### 3. 天气系统

| 天气 | 效果 |
|------|------|
| ☀️ 晴天 | 作物正常生长 |
| 🌧️ 小雨 | 作物生长 +10% |
| ⛈️ 大雨 | 作物生长 +20%，无需浇水 |
| ❄️ 霜冻 | 露天作物停止生长 |

## 作物系统

### 作物分类

游戏中有5种稀有度的作物：

| 稀有度 | 标识 | 作物示例 |
|--------|------|---------|
| 普通 | ⚪ | 萝卜、白菜、土豆 |
| 稀有 | 🟢 | 黄瓜、青椒、梨 |
| 史诗 | 💜 | 葡萄、菠萝、茶树 |
| 传奇 | 🟡 | 樱桃、咖啡、可可、兰花 |

### 作物属性

每种作物都有以下属性：
- **生长周期**：从种植到成熟的天数
- **售价**：收获时的金币收入
- **季节**：适合种植的季节
- **需水量**：每天需要浇水的次数
- **经验值**：收获时获得的经验

### 高价值作物推荐

| 作物 | 生长周期 | 售价 | 利润率 |
|------|---------|------|-------|
| 葡萄 | 14天 | 300 | 300% |
| 樱桃 | 16天 | 450 | 167% |
| 咖啡 | 25天 | 350 | 133% |
| 草莓 | 10天 | 200 | 150% |

## 数据同步

### 自动同步

游戏会自动将你的进度同步到云端：
- 每次收获作物后自动保存
- 每次购买商品后自动保存
- 退出游戏时自动保存
- 手动点击同步按钮

### 手动同步

如果需要强制同步，可以点击界面上的「同步」按钮。

### 跨平台游玩

使用同一账号登录不同平台：
1. 在网页端登录账号
2. 在桌面端登录同一账号
3. 游戏进度会自动同步

### 离线游戏

在没有网络的情况下：
- 游戏仍可正常进行
- 进度会保存到本地
- 联网后自动同步到云端

## 常见问题

### Q: 如何获得更多金币？
A: 
- 种植高价值作物（如葡萄、樱桃）
- 使用肥料提升生长速度和产量
- 建造自动产出建筑（鸡舍、牛棚）

### Q: 体力不足怎么办？
A:
- 等待体力自然恢复（每小时恢复10点）
- 使用体力恢复道具
- 减少频繁操作

### Q: 作物枯萎了怎么办？
A:
- 枯萎的作物无法恢复
- 及时浇水可以避免作物枯萎
- 使用温室可防止恶劣天气影响

### Q: 如何解锁更多土地？
A:
- 在商店购买「土地扩张」道具
- 提升玩家等级可解锁新土地

### Q: 游戏数据丢失怎么办？
A:
- 登录账号可恢复云端存档
- 联系客服寻求帮助

### Q: 如何联系客服？
A:
- 发送邮件至：support@farming-game.com
- 在 GitHub 提交 Issue
