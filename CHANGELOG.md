## 升级指南

### 从v1.0升级到v2.0

1. **备份存档**: 升级前请备份原有的存档文件
2. **安装依赖**: 运行 `pip install -r requirements.txt`
3. **启动游戏**: 使用 `python main.py` 启动新版本
4. **数据迁移**: 游戏会自动转换旧版存档格式
5. **体验新功能**: 推荐尝试新手教程了解新特性

### 系统要求变更
- **新增**: 需要图形界面支持（大多数现代系统都具备）
- **推荐**: 4GB RAM以上，1024x768分辨率以上显示器
- **兼容**: 仍支持纯控制台环境运行
