## 📋 功能清单

### ✅ 已实现的核心功能

#### 1. 身份选择系统
- 6 个可选身份：农民、小镇青年、大学生、海归、个体户、官二代
- 每个身份独特的起始资金、人脉、技能和核心优势
- 精美的身份选择界面，支持响应式设计

#### 2. 主游戏界面
- **每日报告**：显示资产状况、社会地位、个人状态、市场情报
- **行动系统**：
  - 基础行动：工作赚钱、学习提升、休息恢复
  - 商业行动：创建和经营公司（阶层≥4 解锁）
  - 投资行动：理财、房地产（阶层≥5 解锁）
  - 消费行动：购物提升生活质量
  - 查询功能：财务报表、公司状况、城市地图

#### 3. 公司系统
- 创建公司（5 种类型：科技、贸易、餐饮、制造、农业）
- 公司经营和日常管理
- 公司等级、员工、收入、支出、利润追踪

#### 4. 购物系统
- 8 种商品：从经济适用房到私人飞机
- 每种商品提供不同的属性加成
- 根据现金自动判断购买能力

#### 5. 财务报表
- 完整的资产负债表
- 净资产计算
- 社会阶层和财富排名

#### 6. 城市系统
- 4 个可前往城市：广州、深圳、北京、上海
- 每个城市独特的产业和加成效果
- 城市间差异化发展策略

#### 7. 存档系统
- 本地存储自动保存
- 支持手动保存和读档
- 持久化游戏进度

## 🎯 游戏玩法简介

### 游戏目标
从选择的起始身份开始，通过工作、学习、经营公司、投资等方式积累财富，提升社会阶层，最终成为中国百万富翁！

### 社会阶层系统
1. 赤贫阶层（LV1）
2. 温饱阶层（LV2）
3. 小康阶层（LV3）
4. 中产阶层（LV4）- 解锁公司系统
5. 富豪阶层（LV5）- 解锁投资系统
6. 大资本家（LV6）

### 核心策略
- **农民**：专注农业，成本优势
- **小镇青年**：零售业起家，租金优惠
- **大学生**：科技创业，互联网行业
- **海归**：国际贸易，金融投资
- **个体户**：批发市场，制造业
- **官二代**：政府项目，人脉优势（但有腐败风险）
