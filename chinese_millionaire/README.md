# 中国百万富翁 (Chinese Millionaire)

一款 Python 文字模拟经营游戏

## 🎮 游戏特色

- **6 个独特身份**：农民、小镇青年、大学生、海归、个体户、官二代
- **8 个社会阶层**：从赤贫到顶级富豪，阶层跳跃式成长
- **4 个真实城市**：广州、深圳、北京、上海
- **多策略系统**：商业、投资、社交、政治、灰色策略自由组合
- **动态情报**：随身份提升获取更准确的市场信息
- **完整财务报表**：现金流量表、资产负债表、利润表

## 🚀 快速开始

### 运行游戏

```bash
cd chinese_millionaire
python main.py
```

### 系统要求

- Python 3.8+
- Windows/Mac/Linux
- 无需额外依赖

## 📖 游戏玩法

### 1. 选择身份

游戏开始时选择你的起始身份：

- **农民** (¥8,000) - 农业成本 -30%，产量 +25%
- **小镇青年** (¥25,000) - 租金 -40%，小额贷款更容易
- **大学生** (¥50,000) - 科技创业门槛 -50%，融资成功率 +40%
- **海归** (¥200,000) - 国际贸易关税 -30%，外汇手续费 -50%
- **个体户** (¥80,000) - 批发价 -25%，现金流 +30%
- **官二代** (¥150,000) - 政府项目优势，贷款审批更容易

### 2. 核心玩法

- **工作赚钱** - 打工获取初始资金
- **经营公司** - 创建和管理企业
- **投资理财** - 股票、房产、基金
- **拓展人脉** - 社交提升社会地位
- **购物消费** - 提升生活质量

### 3. 阶层晋升

根据净资产自动晋升：

| 阶层 | 净资产要求 | 解锁内容 |
|------|-----------|---------|
| 赤贫阶层 | ¥0 - ¥1 万 | 路边摊、打工 |
| 底层劳工 | ¥1 万 - ¥5 万 | 小生意、夜市摆摊 |
| 小商贩 | ¥5 万 - ¥20 万 | 开小店、倒买倒卖 |
| 小老板 | ¥20 万 - ¥100 万 | 开公司、银行贷款 |
| 中产阶层 | ¥100 万 - ¥500 万 | 房地产、股票基金 |
| 富豪阶层 | ¥500 万 - ¥5000 万 | 并购重组、私募股权 |
| 大资本家 | ¥5000 万 - ¥50 亿 | 跨国贸易、金融杠杆 |
| 顶级富豪 | ¥50 亿+ | 垄断行业、影响政策 |

### 4. 策略组合

选择你的发展策略：

**商业策略**：
- 稳扎稳打（稳健型）
- 薄利多销（走量型）
- 高端定位（利润型）
- 垄断经营（风险型）

**投资策略**：
- 保守理财（年化 5%）
- 股票投机（高风险高回报）
- 房地产投资（稳定增值）
- 风险投资（极高回报）

**社交策略**：
- 低调做人（减少麻烦）
- 高调炫富（吸引机会）
- 广结善缘（人脉增长）
- 攀附权贵（快速晋升）

**政治策略**：
- 合法经营（安全）
- 官商勾结（快速发展）
- 政商旋转门（左右逢源）

**灰色策略** ⚠️：
- 偷税漏税（短期暴利）
- 假冒伪劣（成本大降）
- 内幕交易（稳赚不赔）

### 5. 情报系统

随身份提升获取更准确的情报：

- **低身份**：街头传闻，真假难辨
- **中身份**：市场情报，基本准确
- **高身份**：内幕消息，精准预测
- **顶级身份**：国家机密，上帝视角

## 🏆 游戏目标

- **短期目标**：存活下来，积累第一桶金
- **中期目标**：晋升中产阶层，财务自由
- **长期目标**：成为千亿富豪（¥1000 亿）

## ⚠️ 游戏结束条件

- **破产**：净资产 < 0
- **健康崩溃**：健康值 = 0
- **入狱**：腐败值 = 100（官二代专属风险）
