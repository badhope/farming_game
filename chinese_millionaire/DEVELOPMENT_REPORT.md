### 2. 八层社会阶层

```
1. 赤贫阶层     (¥0 - ¥1 万)      → 路边摊、打工
2. 底层劳工     (¥1 万 - ¥5 万)    → 小生意、夜市摆摊
3. 小商贩       (¥5 万 - ¥20 万)   → 开小店、倒买倒卖
4. 小老板       (¥20 万 - ¥100 万) → 开公司、银行贷款
5. 中产阶层     (¥100 万 - ¥500 万) → 房地产、股票基金
6. 富豪阶层     (¥500 万 - ¥5000 万) → 并购重组、私募股权
7. 大资本家     (¥5000 万 - ¥50 亿) → 跨国贸易、金融杠杆
8. 顶级富豪     (¥50 亿+)         → 垄断行业、影响政策
```

## 🎯 核心设计理念

### 1. 阶层跳跃型成长
- 不是线性增长，而是**阶梯式跨越**
- 每突破一个阶层，解锁全新玩法
- 从温饱到财务自由，真实社会体验

### 2. 情报不对称
- **低身份**：信息模糊、真假难辨
- **高身份**：信息准确、内幕消息
- 反映现实：**信息就是金钱**

### 3. 多策略组合
- 没有"唯一正确"的玩法
- 不同策略组合产生不同结果
- 鼓励多周目体验

### 4. 风险与回报
- 高风险不一定高回报
- 灰色策略可能一夜暴富，也可能锒铛入狱
- 教会玩家**风险管理**

### 5. 真实中国元素
- 4 个真实城市原型
- 中国特色商业文化
- 人情世故、关系网络

## 💡 游戏技巧指南

### 新手建议

**初期 (赤贫 - 小商贩)**:
1. 选择农民或小镇青年，启动资金虽少但加成实用
2. 努力工作积累第一桶金
3. 不要急于投资，先保证生存
4. 多学习提升技能

**中期 (小老板 - 中产)**:
1. 创建公司，稳定收入来源
2. 投资房产，保值增值
3. 拓展人脉，提升社会地位
4. 谨慎使用灰色策略

**后期 (富豪 - 大资本家)**:
1. 资本运作，钱生钱
2. 并购重组，快速扩张
3. 影响政策，制定规则
4. 注意风险管理

### 高手技巧

1. **身份选择**:
   - 速通：海归 (资金多，发展快)
   - 挑战：农民 (白手起家)
   - 平衡：个体户 (稳健型)

2. **策略组合**:
   - 稳健流：稳扎稳打 + 保守理财 + 低调做人
   - 激进流：垄断经营 + 风险投资 + 高调炫富
   - 灰色流：官商勾结 + 偷税漏税 + 内幕交易

3. **城市选择**:
   - 创业：深圳 (科技加成)
   - 从政：北京 (政府项目)
   - 金融：上海 (金融中心)
   - 贸易：广州 (商贸发达)
