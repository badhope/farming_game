# 农场模拟器 Web 版 - 详细开发计划

## 📋 项目概述

### 1.1 项目背景

本项目是一款 AI 驱动的农场模拟游戏，正在从桌面版迁移到 Web 平台。桌面版（v3.x）拥有完整的游戏逻辑，包括：
- 完整的种植系统（10+ 作物）
- 天气和季节系统
- 经济系统（购买种子、出售作物）
- 成就系统
- AI 助手功能
- 农场分析功能

当前已完成：
- FastAPI 后端框架
- 核心游戏逻辑集成
- 基础 API 接口（20+ 个）
- React + TypeScript 前端基础

### 1.2 技术栈

| 层级 | 技术 | 版本要求 |
|------|------|----------|
| 后端框架 | FastAPI | >= 0.100 |
| 后端语言 | Python | 3.8+ |
| 数据库 | SQLite | 3.x |
| ORM | SQLAlchemy | 2.0+ |
| 前端框架 | React | 18+ |
| 前端语言 | TypeScript | 5.0+ |
| 构建工具 | Vite | 5.0+ |
| UI 库 | Ant Design | 5.x |
| HTTP 客户端 | Axios | 1.x |
| 状态管理 | React Context | 内置 |
